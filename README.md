# vue-project-docs

基于 vitepress 搭建的docs ，支持集成到 vue 2/3 项目中。

## 安装

```
pnpm install
```

### 项目启动

```
pnpm run serve
```

### 项目打包

```
pnpm run build
```

## 线上

[项目预览](https://you-hei-mi.github.io/vue-project-docs/)

[项目文档](https://zhuanlan.zhihu.com/p/680310703/)

### 文档启动

```
pnpm run docs:dev
```

### 文档打包

```
pnpm run docs:build
```

### 文档部署

```
pnpm run docs:deploy
```
